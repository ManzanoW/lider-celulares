# Líder Celulares — Landing Page

Landing page profissional desenvolvida para a **Líder Celulares**, assistência técnica autorizada de celulares, tablets e iPads localizada em Santos/SP.

## Sobre o projeto

O objetivo foi criar uma página de alta conversão para uma loja física com mais de 414 avaliações no Google e nota 4.9★, trazendo presença digital profissional com foco em geração de contatos via WhatsApp.

## Funcionalidades

- **Navbar fixo** com blur ao rolar e menu hamburguer responsivo
- **Hero** com logo e foto real da fachada da loja
- **Seção de Serviços** em grid responsivo
- **Seção "Por que escolher a Líder"** com cards, ícones Lucide e efeito hover colorido
- **Logos reais das marcas** atendidas (Samsung, Motorola, Xiaomi, Asus, LG, Apple)
- **Avaliações reais do Google** com badge de nota 4.9★
- **Formulário de contato** + link direto para WhatsApp
- **Google Maps embed** com efeito grayscale → colorido no hover
- **Botão flutuante do WhatsApp** com reposicionamento automático ao chegar no footer
- **SEO completo** com meta tags, Open Graph, Twitter Card e schema.org LocalBusiness
- **Totalmente responsivo** — mobile, tablet e desktop

## Tecnologias

- [Next.js 15](https://nextjs.org/) — App Router
- [React 19](https://react.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Lucide React](https://lucide.dev/) — ícones
